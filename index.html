Immaculate Line

This is a project that was just recently formulated. The idea is similar to that of the Immaculate Grid, a popular online web game involving NFL trivia. The idea of this game is similar: given a starting NFL player and ending NFL player, create a list of players to connect them, using those players' respective teammates.
